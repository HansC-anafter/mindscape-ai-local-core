# 目標客群輪廓 (Target Audience Persona)

本文件定義了「自律神經系統」內容活動的目標受眾。所有內容（文字、視覺）的風格、深度與語氣都應以此為基準。

### 主要輪廓：高知健康探索者

*   **人口統計特徵**:
    *   **年齡**: 28-45 歲
    *   **職業**: 知識工作者、專業人士、教育者、健康產業從業人員。
    *   **教育水平**: 大學或以上學歷。

*   **心理特徵與價值觀**:
    *   對身心健康、個人成長有強烈興趣。
    *   不僅僅滿足於表面建議，傾向於了解事物背後的科學原理。
    *   重視證據和數據，但同時也對整合性的身心實踐（如瑜伽、冥想）抱持開放態度。
    *   生活節奏快，壓力較大，正在積極尋找有效的壓力管理與健康改善方法。

*   **閱讀偏好**:
    *   偏好清晰、結構化、有深度的內容。
    *   能夠理解基礎的科學術語，但反感過度學術化、晦澀難懂的長篇大論。
    *   喜歡透過資訊圖表、簡潔的視覺化來吸收複雜資訊。
    *   對「 actionable insights」（可執行的建議）有高度需求。

*   **內容溝通策略**:
    *   **語氣**: 專業、權威、可信賴，同時帶有啟發性與同理心。
    *   **深度**: 內容應基於科學研究，但以易於理解的語言重新詮釋。
    *   **切角**: 強調科學知識與日常生活的連結，提供讀者可以應用在自己身上的觀點或方法。
