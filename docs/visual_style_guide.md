# 視覺風格指南 (Visual Style Guide)

本文件為「自律神經系統」內容活動定義了統一的視覺語言，以確保所有圖像風格一致，建立專業的品牌形象。

### 1. 核心原則

*   **專業信賴**: 視覺需傳達科學感、精確性與可信度。
*   **寧靜療癒**: 畫面應帶來平靜、專注、療癒的感受。
*   **啟發探索**: 圖像應能引發觀者對身心奧秘的好奇心。

### 2. 色彩計畫

*   **主色調**:
    *   深藍 (#0A2A4E): 代表深度、科學、沉靜。
    *   科技藍 (#3E7FFF): 代表智慧、清晰、科技感。
*   **輔助色**:
    *   金色 (#FFC700): 代表能量、啟發、價值。
    *   白色/淺灰 (#F5F5F5): 用於背景，提供呼吸感。
*   **避免**: 避免使用過於刺眼、飽和度過高的顏色組合。

### 3. 圖像主題與風格

*   **類型一：抽象科學藝術**
    *   **描述**: 神經元網絡、腦波圖、DNA 螺旋等元素的抽象化、藝術化呈現。
    *   **風格**: 簡潔的線條、點狀連結、微光效果，具備科技美學。
    *   **範例**: 類似高品質科學期刊封面或數位藝術作品。

*   **類型二：隱喻性靜物/風景**
    *   **描述**: 平靜的湖面、森林中的光線、生長中的植物、水中的漣漪等。
    *   **風格**: 構圖簡潔、乾淨，光線柔和，氛圍寧靜，能引發觀者對內在平衡的聯想。
    *   **範例**: 高品質的自然攝影，帶有哲思感。

*   **類型三：簡潔資訊圖表**
    *   **描述**: 用於解釋特定概念（如：交感 vs 副交感神經）。
    *   **風格**: 使用本指南定義的色彩計畫，設計清晰、極簡的圖標和流程圖。

### 4. 圖像來源與版權

*   **首選來源**:
    *   **[Unsplash](https://unsplash.com/)**: 使用 Unsplash License 的高品質圖片。
    *   **[Pexels](https://www.pexels.com/)**: 使用 Pexels License 的圖片。
    *   **自創內容**: 使用 AI 繪圖工具或設計軟體，根據本指南創作原創圖像。
*   **版權追蹤**:
    *   **強制要求**: 所有使用的圖片，必須在 `ig_image_prompts.txt` 檔案中對應的貼文區塊記錄其**來源 URL**。
    *   **目標**: 確保每一張發布的圖片都 100% 合規，杜絕任何版權風險。
*   **禁止**: 絕對禁止使用 Google 圖片搜索等來源不明、版權不清的圖片。
