| Scenario family | What continuity is being governed | Public examples |
|-----------------|-----------------------------------|-----------------|

| **Agentic Workflow Sidecar** | Deliberation, memory packet routing, governed writeback across runs | public deep dive coming soon |

These are **scenario families**.

Mindscape Engine is defined by its primitives:

- governance context
- meeting runtime
- `TaskIR`
- `SpatialSchedulingIR`
- governed memory fabric
- runtime receipts and writeback
- world summary and reflection
