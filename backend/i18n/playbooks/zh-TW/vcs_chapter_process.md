# 處理章節

為影片專案中的所有章節產生資產。

## 目標

處理 Video Chapter Studio 專案中的章節，產生縮圖、分析品質，並執行領域特定的擴展分析。

## 輸入需求

- **project_id**：要處理的專案 ID
- **chapter_ids**（選填）：特定章節 ID（預設為全部）
- **options**（選填）：處理選項
  - `extract_thumbnails`：產生縮圖（預設：true）
  - `analyze_quality`：執行品質分析（預設：true）
  - `run_extensions`：執行領域擴展分析（預設：true）

## 處理流程

1. **載入專案**：擷取專案和章節資料
2. **擷取縮圖**：為每個章節產生開始/中間/結束縮圖
3. **分析品質**：檢查影片品質指標（解析度、光線、動態）
4. **執行擴展**：執行領域特定分析（如姿態估計）
5. **彙整結果**：編譯處理結果

## 輸出

- **processed_chapters**：已處理章節結果陣列
- **overall_status**：處理完成狀態

## 注意事項

- 處理以章節為單位執行，可平行處理
- 擴展分析取決於已安裝的領域擴展
- 完整品質分析需要 `ffprobe`
