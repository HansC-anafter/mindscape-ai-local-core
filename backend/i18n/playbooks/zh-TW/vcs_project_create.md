# 建立章節工作室專案

從影片來源建立新的章節化專案。

## 目標

透過匯入各種來源的影片（本地檔案、URL 或 YouTube）來初始化 Video Chapter Studio 專案，並準備進行章節分割。

## 輸入需求

- **video_source**: 來源類型（`file`、`url`、`youtube`）
- **video_ref**: 影片參考（檔案路徑、URL 或 YouTube ID）
- **project_name**（選填）：專案名稱
- **metadata**（選填）：額外的專案中繼資料，包含領域擴展

## 處理流程

1. **驗證輸入**：確認影片來源和參考有效
2. **匯入影片**：下載/複製影片並擷取中繼資料
3. **建立專案紀錄**：將專案儲存至資料庫
4. **回傳結果**：回傳已建立的專案資訊

## 輸出

- **project_id**：建立專案的唯一識別碼
- **video_metadata**：擷取的影片資訊（時長、解析度、幀率）
- **status**：專案建立狀態

## 注意事項

- YouTube 影片將在允許時進行下載
- 影片中繼資料擷取需要 `ffprobe` 系統工具
- 大型影片可能需要較長的匯入時間
