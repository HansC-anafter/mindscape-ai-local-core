# 檔案寫入

## 描述

將內容寫入到工作區儲存目錄中的檔案。

## 使用情境

- 生成檔案並寫入內容
- 儲存文字資料到檔案
- 創建配置檔案或文件檔案

## 輸入參數

- **file_path** (必需): 相對於工作區儲存目錄的檔案路徑
- **content** (必需): 要寫入的檔案內容
- **encoding** (選填): 檔案編碼，預設為 utf-8

## 輸出

- **file_path**: 寫入的檔案路徑
- **size**: 檔案大小（位元組）
- **success**: 是否成功寫入

## 範例

寫入一個測試檔案：

```json
{
  "file_path": "test_file.txt",
  "content": "這是一個測試檔案",
  "encoding": "utf-8"
}
```
