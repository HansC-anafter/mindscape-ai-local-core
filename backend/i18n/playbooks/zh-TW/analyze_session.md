# 沖煮會話分析

取得沖煮會話的詳細穩定度指標與表現評分。

## 概述

此 Playbook 分析擷取的重量曲線資料，提供關於手沖技術的可操作洞察。

## 分析內容

### 段落偵測
- **悶蒸**：初始潤濕階段（約 0-40 秒）
- **主注水**：主要萃取階段（約 40-95 秒）
- **末段**：收尾階段（約 95-110 秒）

### 各段落指標
- 平均流速（g/s）
- 流速標準差（穩定度指標）
- 段落狀態：良好 / 不穩 / 過快 / 過慢

### 總分
- 0-100 分，基於穩定性與一致性
- 等級：A（90+）、B（80-89）、C（70-79）、D（60-69）、F（<60）

### 偏差偵測
- 流速不穩（高標準差）
- 注水中斷
- 流速超出目標範圍

## 輸出

分析結果包含：
- 總分與等級
- 各段落指標
- 偵測到的偏差列表
- 關鍵重點（最多 3 項）
